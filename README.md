# こうかとんマスターズ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ライフ制、PvPのカードゲーム
　ライフを削りきるもしくは相手の山札がなくなったら勝ち
* デッキ枚数20枚、5種4枚、使ったカード墓地
* 参考

## ゲームの遊び方
* ランダムで先攻後攻が決まり、はじめの手札が3枚
* 毎ターンマナと手札が1つずつ増える(先攻1ターン目は手札が増えない)

* アタックすると指定の数をライフを削るかタップしているクリーチャーにアタックできる(出たターンアタック不可)アタック後タップする

* アタックしてないクリーチャーはブロックできる
